## Getting Started

Pra clonar o repo & rodar tudo:

```bash
git clone https://github.com/ArthurPimentaFatec/soromaps.git
# depois de clonar vc entra no projeto
cd soromaps
# roda o vscode pra abrir direto na pasta
code .
# e starta o servidor direto pelo cmd pq pelo vscode nao funciona nos pc da fatec
npm run dev
```

```bash
# pra rodar o backend entra na pasta do backend
cd src/services/soromaps
# starta o backend
dotnet run
```
